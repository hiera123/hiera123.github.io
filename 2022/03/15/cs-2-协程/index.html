定义

  1. Coroutines allow one to spread[传播] the exuction[执行] of code over multiple frames using the yield statement.

  2. yield return new WaitForSeconds(2) 等待指定时间之后在执行

  1. yield return null; 等待至下一帧继续执行

  1. yield return StartCoroutine(DoSoming()); 等待对方协程结束后再继续执行

  2. StopCoroutine(DoSoming()) 先获取当前协程的引用，再停止协程
